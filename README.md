# ✅ To Do

Projeto desenvolvido como estudo no curso de nível intermediário de React da plataforma Rocketseat.

## 🚀 Tecnologias

- React
- TypeScript
- CSS / TailwindCSS
- Vite
- HTML
- React Router
- localStorage
- class variance authority (CVA)

## 🎯 Funcionalidades

- Criar e salvar atividades.
- Listar atividades criadas.
- Marcar como "feito" atividades existentes.
- Exibir quantas atividades já foram concluídas.
- Exibir quantas atividades foram criadas.
